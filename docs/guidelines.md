# Conventional Commits para AI

## Formato

```text
<tipo>(<escopo-opcional>): <acao curta>
```

## Regras

- use verbo no infinitivo
- primeira linha até 72 caracteres
- sem ponto final no título
- um commit = uma intenção

## Tipos comuns

- `feat`
- `fix`
- `docs`
- `refactor`
- `test`
- `chore`
- `ci`
- `build`
