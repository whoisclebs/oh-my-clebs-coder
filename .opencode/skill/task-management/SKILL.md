# Task Management Skill

## Objetivo

Dar uma estrutura leve para quebrar trabalho complexo em tarefas pequenas e verificáveis.

## Estrutura

Use `.tmp/tasks/active/<feature>/` para tarefas ativas e `.tmp/tasks/completed/<feature>/` para histórico.

## Arquivos

- `task.json`
- `subtask_01.json`, `subtask_02.json`
